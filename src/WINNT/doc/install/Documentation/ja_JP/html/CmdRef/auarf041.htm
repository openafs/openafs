afszcm.cat

目的

キャッシュ・マネージャーのデバッグのためのエラー・メッセージ・カタログ。

構文

afszcm.cat ファイルは、キャッシュ・マネージャーのためのメッセージ・カタログです。 fstrace dump コマンド・インタープリターは、このファイルを標準の UNIX カタログ・ユーティリティーと共に使用して、 fstrace トレース・ログにトレースを書き込む際に、キャッシュ・マネージャーの操作コードを文字列に変換します。これによって、ログがより読みやすくなります。

このファイルの基本的な位置は /usr/vice/etc/C/ ディレクトリーです。 NLSPATH および LANG 環境変数が必要に応じて設定されている場合は、これを別のディレクトリーに置くことができます。
