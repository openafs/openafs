kas forgetticket

目的

発行者のすべてのチケットを破棄します。

構文 

kas forgetticket [-all]  [-help]
    
kas f [-a]  [-h]

構文

kas forgetticket コマンドは、ローカル・マシンのカーネル・メモリーに保管されている発行者のすべてのチケットを廃棄します。このチケットには、ユーザーが認証した際の各セルからの AFS サーバー・チケット、およびユーザーが現在の kas セッション中に (セッションに入る時、または kas getticket コマンドを使用することによって) 獲得したすべてのチケットが含まれます。

オプション

-all
すべてのチケットを破棄します。この引き数は、明示的にコマンドのデフォルトの振る舞いを起動します。

-help
このコマンドのオンライン・ヘルプを出力します。これ以外の有効なオプションはすべて無視されます。

例

以下のコマンドは、すべての発行者のチケットを破棄します。 

   % kas forgetticket

必要となる特権

なし。パスワードは不要です。
