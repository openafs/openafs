ThisCell (サーバー・バージョン)

目的

サーバー・マシンのセル・メンバーシップを定義します。

構文

サーバー・バージョンの ThisCell ファイルには、サーバー・マシンが属するセルの完全なインターネット・ドメイン・スタイルの名前 (たとえば、abc.com) が定義されています。このファイルは、すべての AFS サーバー・マシン上の /usr/afs/etc ディレクトリーに常駐している必要があります。

このファイルは ASCII 形式で、1 行に 1 つの文字列で作成されています。このファイルの初期バージョンは、セルの最初のファイル・サーバー・マシンがインストールされるときに bos setcellname コマンドで作成されます。追加サーバー・マシンのインストール時にこのファイルをこの追加マシンにコピーする方法については、AFS インストールの手引き を参照してください。

このファイルは、セル名を変更したときにだけ編集してください。セル名を変更すると、多くの構成情報を変更しなければならないので、お勧めはしません。特に、認証サーバーは、このファイルにあるセル名と各ユーザーおよびサーバーのパスワードを組み合わせて、暗号化キーを生成してからデータベースに記するため、セル名を変更すると、認証データベース全体を再構築しなければなりません。

サーバー・バージョンの ThisCell ファイルは、各 AFS クライアント・マシン上の /usr/vice/etc ディレクトリーにあるクライアント・バージョンとは異なります。サーバー・マシンがクライアントとして実行されている場合には、同じマシンのサーバーおよびクライアント・バージョンのファイルに異なるセルが指定されていても構いません。ただし、この構成の場合、より複雑になります。
