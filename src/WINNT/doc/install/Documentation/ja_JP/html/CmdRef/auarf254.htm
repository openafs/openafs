vos apropos

目的

キーワード文字列を含む各ヘルプ項目を表示します。

構文 

vos apropos -topic <help string>  [-help]
    
vos ap -t <help string>  [-h]

構文

vos apropos コマンドは、 vos コマンドの名前または簡単な説明の中で、 -topic 引き数に指定された文字列を含むコマンドのオンライン・ヘルプの最初の行を表示します。

コマンドの構文を表示するには、vos help コマンドを使用します。

オプション

-topic
検索するキーワード文字列を指定します。頭字語 VLDB 以外は、すべて小文字を使用してください。複数の単語を入力する場合には、各単語を二重引用符 ("") か、他の区切り文字で囲んでください。

-help
このコマンドのオンライン・ヘルプを出力します。これ以外の有効なオプションはすべて無視されます。

出力

コマンドのオンライン・ヘルプ項目の最初の行には、コマンドの名前と簡単な機能説明があります。このコマンドを実行すると、コマンド名または最初の行に -topic 引き数に指定された文字列がある vos コマンドの最初の行が表示されます。

例

次のコマンドは、名前または説明部分に lock という言葉がある vos コマンドをすべて表示しています。 

   % vos apropos lock
   lock: lock VLDB entry for a volume
   unlock: release lock on VLDB entry for a volume
   unlockvldb: unlock all the locked entries in the VLDB

必要となる特権

なし。
