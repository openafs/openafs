uss help

目的

指定された uss コマンドの構文、またはすべての uss コマンドの簡単な説明を表示します。

構文 

uss help [-topic <help string>+]  [-help]
    
uss h [-t <help string>+]  [-h]

構文

uss help は、-topic 引き数で指定されたコマンド操作コードについて、完全なオンライン・ヘルプ項目 (簡単な説明と構文情報) を表示します。-topic 引き数を省略すると、各 uss コマンドのオンライン・ヘルプ項目の最初の行 (名前と簡単な説明) が表示されます。

名前または説明部分に特定のキーワードを含む uss コマンドを表示する場合には、 uss apropos コマンドを使用してください。

オプション

-topic
各コマンドの完全なオンライン・ヘルプ項目が表示されます。コマンド名から uss を除いた部分を指定してください。(たとえば、uss bulk ではなく、bulk と指定します。) この引き数を省略すると、各 uss コマンドの簡単な説明が表示されます。

-help
このコマンドのオンライン・ヘルプを出力します。これ以外の有効なオプションはすべて無視されます。

出力

各 uss コマンドのオンライン・ヘルプ項目は、次のように 2、3 行から構成されます。

  • 最初の行には、コマンド名および機能の説明が表示されます。
  • 次の行には、コマンドの別名 (ある場合) が表示されます。
  • 最後の行は、Usage という文字列で始まり、コマンドのオプションが一定の順序で表示されます。オンライン・ヘルプの項目では、この文書の解説ページと同じ記号 (大括弧など) が使用されています。

例

次の例では、uss bulk コマンドのオンライン・ヘルプ項目を表示します。 

   % uss help bulk
   uss bulk: bulk input mode
   Usage: uss bulk -file <bulk input file> [-template <pathname
   of template file>] [-verbose] [-cell <cell name>] [-admin
   <administrator to authenticate>] [-dryrun] [-skipauth] [-overwrite]
   [-pwexpires <password expires in [0..254] days (0 => never)>] [-pipe]
   [-help]

必要となる特権

なし。
