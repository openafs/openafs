pts apropos

目的

キーワード文字列を含む各ヘルプ項目を表示します。

構文 

pts apropos -topic <help string>  [-help] 
   
pts ap -t <help string>  [-h]

構文

pts apropos コマンドは、 -topic 引き数が指定する文字列をコマンド名または説明部分に含む、任意の pts コマンドのオンライン・ヘルプ項目の最初の行を表示します。

コマンドの構文を表示する場合には、 pts help コマンドを使用してください。

オプション

-topic
検索するキーワード文字列を、英小文字のみで指定します。その文字列が 2 語以上である場合は、それを、二重引用符 ("") あるいはその他の区切り文字で囲んでください。

-help
このコマンドのオンライン・ヘルプを出力します。これ以外の有効なオプションはすべて無視されます。

出力

コマンドのオンライン・ヘルプ項目の最初の行には、コマンドの名前と簡単な機能説明があります。このコマンドを実行すると、 pts コマンドの最初の行が表示されます。 -topic 引き数が指定する文字列は、このコマンド名または最初の行に含まれています。

例

次のコマンドは、コマンド名または簡単な説明の中に、 create という単語が含まれている pts コマンドを、すべてリスト表示するものです。 

   % pts apropos create
   creategroup: create a new group
   createuser: create a new user

必要となる特権

なし。
