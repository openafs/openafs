管理解説書

UserList

目的

特権管理者を定義します。

構文

UserList ファイルには、ローカル・サーバー・マシンまたはそのマシンに格納されているボリュームに及ぼす特権のある bos、vos、および backup コマンドの発行が許可されているシステム管理者の AFS ユーザー名が記録されています。このファイルはすべてのサーバー・マシンの /usr/afs/etc ディレクトリーになければなりません。

UserList ファイルは ASCII 形式ですが、テキスト・エディターで更新しないでください。編集する場合には、次のように、bos コマンド・セットから適切なコマンドを使用してください。

  • bos adduser コマンド。ファイルにユーザーを追加します。
  • bos listusers コマンド。ファイルの内容を表示します。
  • bos removeuser コマンド。ファイルからユーザーを削除します。

別のサーバー・マシンの UserList ファイルにある管理者を表示することは理論的には可能ですが、予期しない認証エラーが起きる可能性があるので、お勧めはしません。米国版の AFS を実行し、更新サーバーを使用して /usr/afs/etc ディレクトリーの内容を配布しているセルでは、システム制御マシンに保管されているファイルのコピーだけを編集します。国際版の AFS を実行しているセルの場合には、各サーバー・マシンごとにファイルを編集します。
