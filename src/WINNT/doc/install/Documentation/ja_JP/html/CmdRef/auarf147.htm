fs help

目的

指定された fs コマンドの構文を表示するか、すべての fs コマンドの機能説明をリストします。

構文 

fs help [-topic <help string>+]  [-help]
   
fs h [-t <help string>+]  [-h]

構文

fs help コマンドは、-topic 引き数によって指定された各コマンドの命令コードについての、完全なオンライン・ヘルプ項目 (概説および構文ステートメント) を表示します。 -topic 引き数が省略された場合は、すべての fs コマンドのオンライン・ヘルプ項目の最初の行 (名前および概説) が出力されます。

指定されたキーワードを名前や概説に含んでいる fs コマンドをすべて表示するには、 fs apropos コマンドを使用します。

オプション

-topic
各コマンドの完全なオンライン・ヘルプ項目が表示されます。コマンド名の fs の部分を省略し、命令コードだけを指定します ( たとえば、fs setacl ではなく setacl を指定)。この引き数が省略されると、すべての fs コマンドの概説が出力されます。

-help
このコマンドのオンライン・ヘルプを出力します。これ以外の有効なオプションはすべて無視されます。

出力

各 fs コマンドのオンライン・ヘルプ項目は、次の 2 行または 3 行から構成されます。

  • 最初の行には、コマンド名および機能の説明が表示されます。
  • 次の行には、コマンドの別名 (ある場合) が表示されます。
  • 最後の行は、Usage という文字列で始まり、コマンドのオプションが一定の順序で表示されます。オンライン・ヘルプの項目では、この文書の解説ページと同じ記号 (大括弧など) が使用されています。

例

次のコマンドは、fs setacl コマンドのオンライン・ヘルプ項目を表示します。 

   % fs help setacl 
   fs setacl: set access control list
   aliases: sa 
   Usage: fs setacl -dir <directory>+
   -acl <access list entries>+ [-clear] [-negative] [-help]

必要となる特権

なし。
