릴리스 노트

소개

개요

AFS(R)는 기업 내에서 정보에 대한 안전하고 안정적인 액세스를 제공하는 분산 파일 시스템입니다. 파일 서버 기계에 있는 디렉토리와 파일을 어떤 데스크 탑에서도 하나의 파일 시스템으로 통합시킴으로써, AFS는 사용자에게 기계의 영역과 무관한 단일한 파일 공간을 제공합니다.

AFS는 여러 이점을 제공합니다. 클라이언트측의 캐쉬와, 여러 파일 서버에 걸쳐 자주 액세스되는 데이터의 복제를 허용함으로써 파일의 가용성을 높여줍니다. AFS는 파일 공간에 저장된 정보를 보호하는 강력한 보안 기능을 제공합니다. 추가로, AFS는 확장가능합니다. 서버와 클라이언트 기계가 기존의 기계에 거의 영향을 미치지 않으면서 AFS 구성에 무제한적으로 추가될 수 있습니다. 즉, 기업의 성장과 더불어 파일 시스템도 성장이 가능합니다.

Windows용 AFS는 다음 제품을 포함합니다.

  • AFS 서버

    AFS 서버는 Windows NT(R) 기계에서 AFS 서버 프로세스를 실행합니다. Windows NT에 있는 AFS 서버는 AFS 파일 서버, AFS 데이터베이스 서버, AFS 백업 서버 및 AFS 시스템 제어 서버 중 하나 이상으로 구성될 수 있습니다. AFS 서버에는 설정을 돕기 위한 AFS 서버 구성 마법사가 들어 있습니다.

  • AFS 제어 센터

    AFS 제어 센터에는 AFS 셀 관리를 돕기 위해 두 개의 강력한 그래픽 사용자 인터페이스(GUI)를 포함합니다. 즉, 하나 이상의 AFS 서버에서 볼륨 및 서비스의 관리를 이용하는 도구인 AFS 시스템 관리자와, 하나 이상의 AFS 서버에서 사용자와 그룹 계정의 간단한 작성 및 유지보수를 가능하게 하는 도구인 AFS 계정 관리 프로그램이 있습니다.

  • AFS 클라이언트

    AFS 클라이언트는 Windows NT를 실행하는 PC에서 AFS 파일 공간에 대한 액세스를 제공하며, 사용자가 AFS에서 파일과 디렉토리를 관리하게 합니다.

  • AFS 라이트

    AFS 라이트는 AFS 클라이언트가 설치된 또다른 PC로 AFS 요청을 전달하여 Windows 98이나 Windows 95를 실행하는 PC에서 AFS 파일 공간에 대한 액세스를 제공합니다.

릴리스 노트에 대한 정보

이 문서에는 Windows용 AFS, 버전 3.6에 대한 중요한 정보가 들어 있습니다. 이 문서는 설치 전제조건, 제품 주의사항, 특정 제한사항 및 이 릴리스의 제한사항을 요약합니다.

주:Windows용 AFS 사용자는 README.txt 파일을 보고 이 문서를 기록한 이후에 사용가능하게 된 추가적인 Windows용 AFS의 기능을 확인하게 할 수 있습니다. README.txt 파일은 Windows용 AFS가 설치되었을 때 Windows용 AFS 설치\documentation 디렉토리에 설치됩니다.

이 책의 사용자

이 책은 Windows용 AFS에 포함된 제품의 설치, 구성 및 사용에 책임이 있는 시스템 관리자용으로 기록됩니다. 이 책은 독자가 보통 시스템 관리와 AFS 사용에 친숙하다고 가정합니다.
